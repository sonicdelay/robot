### Install
```
sudo apt-get install libxtst-dev libpng++-dev
npm install -g node-gyp
node-gyp rebuild

git clone https://github.com/sonicdelay/robot.git
npm i
npm start
```

http://localhost:5000 or http://< IP >:5000

See also: https://github.com/octalmage/robotjs#building
